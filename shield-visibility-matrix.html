⚡ Real-Time Controls (TSPs)

  • Transaction Security Policies evaluate on streaming events and can block, end session, notify, or alert in real-time
  • TSPs require an ApiEvent or LoginEvent to fire — no event, no enforcement
  • For MCP-routed apps (L4), LoginEvent TSP is the only real-time lever — use gateway IP filtering as a binary kill switch
  • Agentforce (L1) supports agent-scoped policies using BotId field
  • CLI tools show Client as sfdx-toolbelt (generic, not app-specific) — scope with Client.containsIgnoreCase('sfdx')
  • ConnectedAppId field is only populated with Connected App auth — use it to identify specific apps

📋 Delayed Controls (ELOs)

  • Event Log Files are generated daily and available ~24 hours after activity
  • All four surfaces have ELO coverage — auditable but not enforceable in real-time
  • MCP traffic appears in ApiTotalUsage attributed via ConnectedAppId
  • Export ELOs to your SIEM (Splunk, Sentinel) for compliance monitoring
  • Enhanced Event Logs with agent-specific telemetry are Agentforce-only and land in Data Cloud

* Conditional Visibility (L2, L3): Real-time ApiEvent visibility depends entirely on authentication method, not the tool or surface:

• Connected App (session token) → Full ApiEvent visibility + ConnectedAppId populated, TSPs fire normally

• External Client App (ECA JWT) → Zero ApiEvent records, no ConnectedAppId, TSPs cannot enforce — same blind spot as MCP

Client field: The Client field shows generic values like sfdx-toolbelt for CLI tools — it identifies the tool type, not the specific application. To identify specific apps, use ConnectedAppId (only available with Connected App auth).

sf-cli default: The Salesforce CLI uses Connected App authentication by default, so L2 typically has full visibility. However, if configured to use ECA JWT, it would lose real-time ApiEvent coverage.

Key Insight: ECA is Salesforce's recommended auth pattern for AI/MCP integrations, but it trades real-time Shield visibility for improved security architecture. Customers making the ECA choice should understand this trade-off.
